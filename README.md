# Epidemiological Measures

An interactive educational web application that explains fundamental concepts in epidemiology through visualizations, calculators, and quizzes.

## Overview

This presentation-style website teaches epidemiological measures including:
- Counts, rates, and risks
- Incidence and prevalence
- Mortality and case fatality
- Years of Potential Life Lost (YPLL)
- Birth rates and fertility rates
- Life expectancy and lifetime risk

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to `http://localhost:3000`

## Build for Production

```bash
npm run build
npm run preview
```
